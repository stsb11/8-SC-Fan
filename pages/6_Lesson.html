1 Pulse Width Modulation (PWM)

Learn It

  • The GENIE chip uses digital outputs.
  • This means that we can either have the motor on, or the motor off.
  • The problem is, we want to control the speed of the fan, so how can we use on/off to change how fast the motor spins

Research It

  • Conduct a little web research to find out what is mean by Pulse Width Modulation (PWM).

Badge It - Silver

  • Write a program to turn the fan on and off when the Push-To-Make switch (PTM) is pushed and released

Badge It - Gold

  • Write a program to turn the fan on when the PTM is pushed once, and off when pushed again.

Badge It - Platinum

  • Write a program that allows the user to change the speed the fan spins at by using what you have learned about PWM.
